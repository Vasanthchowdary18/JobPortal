# Job Portal Website

## 🔎 Overview

A responsive Job Portal web application built using HTML and CSS. It includes multiple pages such as Home, About, Company, Contact, Login, and Registration, designed to simulate a real-world job portal with a clean and user-friendly interface.

---

## ✨ Features

* 🏡 Home Page
* 📖 About Page
* 🏬 Company Page
* ☎️ Contact Page
* 🔑 Login Page
* 📋 Registration Page
* 🎯 Simple and clean UI
* 🎬 Background media support

---

## ⚙️ Technologies Used

* HTML5
* CSS3

---

## ▶️ How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Locate the `index.html` file.
4. Double-click on `index.html` to open in your browser.

### 💻 Using VS Code (Recommended)

1. Open the project in VS Code.
2. Install the **Live Server** extension.
3. Right-click on `index.html`.
4. Click **"Open with Live Server"**.

---

## 📥 How to Clone

1. Go to the GitHub repository:
   https://github.com/Vasanthchowdary18/JobPortal
